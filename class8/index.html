What's The Tech World Look Like?

  • Agile development: Small changes, constant improvement
  • Cloud Apps: No worrying about your own infrastructure
  • DevOps: Continuous compile, test, deploy, run of code
  • Microservices: Separate services, individual upgrades

What Software Gets Used Out There?

  • IDEs: Visual Studio, IntelliJ, Eclipse
  • DevOps: Docker, GitHub / GitLab / Bitbucket, Jenkins / Bamboo / TeamCity, Ansible / Puppet / Chef
  • Languages: JavaScript (and frameworks), Python, Java, C#, Go, Kotlin
